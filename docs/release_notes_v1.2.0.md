The class update — **a new class, and a fairer fight**.

## New class: Echo

Press **J** to **Rewind**: snap back to where you were 2 seconds ago, past momentum included. A ghost echo marks the destination for a beat before you go — chasers can read it, so a predictable rewind gets met at the door. The skill is making your past position a good future one. (9s cooldown; getting tagged during the wind-up stands.)

## Balance pass

- **All classes now run at the same speed.** Nobody wins a flat chase by stat sheet alone — class identity lives in jump, weight, and ability.
- **Stun Pulse** radius up 120 → 150, and the blast now fills the whole affected disc and holds for a beat, so everyone can see exactly who was in range.
- **Ability key moved from Q to J** — your left hand runs, your right hand jukes.
- **New ability HUD**: bottom-left, styled like the menus, with a J keycap and a proper cooldown bar that lights up teal when ready.

Same join codes. Windows: SmartScreen → *More info → Run anyway*. macOS: right-click → Open the first time.
