The big one: **Slippington now plays right in your browser** — no download, just a link.

## Play in your browser

Go to **https://shawnli14.github.io/slippington/** and you're in: same join codes, the same low-latency peer-to-peer tag, nothing to install. Create a game, share the code, and your friends just click the link. Works best in Chrome or Firefox on desktop. The downloadable Windows and macOS builds are still here below if you prefer them.

## The slush is gone

The endgame flood that rose from the bottom of the map and slowed everyone down has been removed — the final seconds are now a clean, all-out scramble.

## Fixes

- **No more surprise disconnects.** Fixed players getting kicked with "host has left the game" the instant a match started, and hardened connections so a hiccup on the matchmaking server no longer drops a game in progress.
- **Full-screen sky.** Fixed a gray strip down the edge of the screen on browsers and windows that aren't exactly 16:9.

Same join codes. Windows: SmartScreen → *More info → Run anyway*. macOS: right-click → Open the first time.
