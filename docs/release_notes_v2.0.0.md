Slippington got a complete makeover. Same fast, peer-to-peer tag — a whole new look.

## A bold new style

Every screen has been redesigned with a chunky, playful "party" look:

- A big, characterful title and redesigned class-select cards — the class you pick pops out so it's obvious what you're bringing into the match.
- Punchy, tactile buttons; a cleaner lobby with the join code front-and-center; and a livelier results screen.
- A refreshed in-match HUD that's bolder but still stays crisp and readable while you're running for your life.

## Backgrounds that actually look like somewhere

The flat, simple backdrops are gone. Maps now sit in layered, atmospheric scenes with real depth:

- **Sky Islands** — smooth floating islands with grassy tops and little trees, drifting clouds, warm sun glow, and a soft haze on the horizon.
- **Frozen Lake** — snow-capped peaks under a shimmering aurora, with drifting snow and rim-lit ridges.
- A gentle vignette and graded skies tie it all together, so the players and platforms always stay the clear focus.

## Notes

- This release is for **Windows and macOS**. The browser version is unchanged for now and will get the new look in a future update.
- Windows: SmartScreen → *More info → Run anyway*. macOS: right-click → *Open* the first time.
